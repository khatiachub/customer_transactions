💱 Retail Sales & Customer Performance Dashboard

✅Project Overview

This project features a comprehensive Business Intelligence (BI) solution developed in Power BI to analyze retail transaction data. The goal is to provide actionable insights into product performance, customer behavior, and regional sales distribution.

↔️Key Features

Customer Profiling: Built dynamic user profile cards showing individual rankings, total transaction values, and activity status.
Product Performance Matrix: Utilized Scatter Charts to visualize the correlation between transaction counts and total revenue per product.
Financial Analysis: Integrated a Tornado Chart to compare Debit vs. Credit transaction distributions across the business.
Geospatial Insights: Developed interactive maps to track sales performance and regional trends across various cities and regions.
Advanced Data Modeling: Implemented DAX measures for ranking (RANKX) and Top N filtering to isolate high-value customers within specific product categories.

🟢Tech Stack

Power BI: Data visualization and dashboarding.
Excel: Data preprocessing and initial Pivot Table analysis.
DAX (Data Analysis Expressions): Complex calculations and custom metrics.
